# Fitness Analytics Dashboard | Power BI Project

## Project Overview
This project presents a Fitness Analytics Dashboard developed using Power BI to analyze gym operations, member information, and financial performance. The dashboard provides insights into business performance, membership distribution, and member demographics, enabling better decision-making through data visualization.

## Tools Used
- Power BI
- Microsoft Excel (Data Preparation and Cleaning)

## Dashboard Features

### Financial Performance Analysis
The dashboard includes key financial metrics such as total revenue, expenses, and profit. It also provides monthly financial trends to monitor business performance over time.

### Membership Analysis
Membership information is analyzed to understand the distribution of active and expired memberships as well as membership categories such as Platinum, Gold, and Silver.

### Member Information and Demographics
The project includes detailed member data including gender, age group distribution, membership status, and trainer assignments. This helps in understanding the customer base and demographic trends.

### Fitness Calculator
An interactive fitness calculator is included to estimate BMI, BMR, and daily calorie requirements based on user inputs such as age, height, weight, and activity level.

## Key Insights
- Overview of total clients and trainers
- Revenue, expenses, and profit analysis
- Membership distribution and status tracking
- Member demographic analysis based on age and gender
- Interactive health and fitness calculations

## Purpose of the Project
The purpose of this project is to demonstrate the use of Power BI for building interactive dashboards that provide meaningful insights from business and fitness-related datasets.

## Author
Data Analytics Project developed as part of a learning portfolio to demonstrate skills in data visualization, dashboard development, and business data analysis using Power BI.
